# TicTacToe App

TicTacToe App is a simple yet engaging game developed using ReactJS. The game features a responsive design, allowing users to enjoy playing TicTacToe on various devices.

## Demo

Play the game online [here](https://game-react-tictactoe.netlify.app/).

## Technologies Used

- ReactJS
- CSS
- JavaScript

## Features

- Responsive design for optimal gameplay across devices
- Intuitive user interface for a seamless gaming experience
- Play against a friend or against the computer

## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/Bikal-Adhikari/react-game.git
   ```

2. Install NPM packages

   ```sh
   npm install
   ```

### Usage

1. Start the development server

   ```sh
   npm start
   ```

2. Open your browser and go to:

   ```sh
   http://localhost:3000
   ```

3. Enjoy playing TicTacToe!
